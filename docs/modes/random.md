# Modo Random (post-MVP o stretch)

> **Estado:** borrador / fuera de MVP salvo que digamos lo contrario

## Idea

Cada partida (o cada X olas) ofrece un subset aleatorio de torres / upgrades, forzando builds distintos.

## Opciones

| Variante | Descripción |
|----------|-------------|
| Random starter | 3–4 torres random al inicio |
| Draft | Elegís N de un pool random |
| Chaos upgrades | Las ramas de upgrade cambian |

## Riesgos

- RNG frustrante si el pool no tiene anti-tank o AoE.
- Mitigación: garantizar al menos 1 torre por rol esencial.

## Decisión

¿Entra en MVP? **No** (recomendado).
